# Import/Export

Agin Requests can import a collection from the following formats:
- **OpenAPI** v3 JSON or YAML files *(coming soon!)*
- **Thunder Client** collections *(coming soon!)*
- **Postman** format *(coming soon!)*
- **Insomnia** JSON file *(coming soon!)*

### Importing cURL

Agin Requests can also import individual HTTP requests saved as cURL commands.

<Image src="/images/curl.png" width={200} height={235} style={{ marginTop: '12px', borderRadius: '5px' }} />

- Click the <IconChevronDown style={{ display: 'inline' }} size={18} stroke={3} /> **arrow** next to **New Request** button and choose **import from cURL**
- Paste your cURL command
