# Path Variables

You can use <IconBraces style={{ display: 'inline', transform: 'translateY(-2px)' }} size={18} /> **Path Variables** in your requests to make them more dynamic. Path variables are defined in the URL path and are in braces `{}`. For example, if you have a URL like `https://api.example.com/users/{userId}`, you can insert `{userId}` and then change its value in the request editor.
